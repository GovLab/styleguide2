The code for the slider consists in using a .slider element to wrap multiple slide components. That will automatically trigger the JS and all the extra functionality and styles used by slick.js. We are extending Slick with our own visual styles.


              <div class="slider">
                <div class="slide">...</div> <!-- Slide Component -->
                <div class="slide">...</div> <!-- Slide Component -->
                <div class="slide">...</div> <!-- Slide Component -->
                ...
              </div>

The .table-sortable element uses List.js for sorting and fuzzysearch. This example uses three classes, .item__name, .item__category, and .item__date.


                  <div  id="items"> <!-- Wrap the table with an id that will be referenced by list.js -->
                    <table class="table-sortable table-sortable--main-color">
                    <!-- .table-sortable--fixed -->
                      <thead>
                        <th class="table-sortable__head-cell"> </th>
                        <th class="table-sortable__head-cell item__name">
                          <span>Name</span>
                          <div  class="table-sortable__control-group">
                            <span class="table-sortable__control table-sortable__control--simple js-open-table-search" data-target="#item__name--input"><i class="material-icons">search</i></span>
                            <span class="table-sortable__control table-sortable__control--simple sort" data-sort="item__name"><i class="material-icons">keyboard_arrow_down</i></span>
                          </div>
                          <form class="table-sortable__search ">
                            <input id="item__name--input" class="fuzzy-search" type="text" placeholder="Filter by text">
                            <button type="submit" class=""><i class="material-icons">close</i></button>
                          </form>
                        </th>
                        <th class="table-sortable__head-cell item__category">
                          <span>Category</span>
                          <div  class="table-sortable__control-group">
                            <span class="table-sortable__control table-sortable__control--simple sort" data-sort="item__category"><i class="material-icons">keyboard_arrow_down</i></span>
                          </div>
                        </th>
                        <th class="table-sortable__head-cell item__date">
                          <span>Date</span>
                          <div  class="table-sortable__control-group">
                            <span class="table-sortable__control table-sortable__control--simple sort" data-sort="item__date"><i class="material-icons">keyboard_arrow_down</i></span>
                          </div>
                        </th>
                        <th class="table-sortable__head-cell"> </th>
                      </thead>
                      <tbody class="list"> <!-- class required by list.js -->
                        <tr class="item">
                          <td></td>
                          <td class="item__name" data-target="Colliers International"><a href="#">Colliers International</a></td>
                          <td class="item__category">Housing/Real Estate</td>
                          <td class="item__date" data-timestamp="1898">1898</td>
                          <td></td>
                        </tr>
                        <tr class="item">...</tr> <!-- table row -->
                        <tr class="item">...</tr> <!-- table row -->
                      </tbody>
                    </table>
                  </div>
